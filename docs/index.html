FAQ
Why is EYWA better
than similar projects?
EYWA is a middle layer for all blockchains that solves an interoperability issue and opens limitless
opportunities for operations among different chains: cross-chain dApps development,
asset management, etc.

EYWA has the following features
that makes it stand out:
EYWA is fully decentralized cross‑chain data bridges enabling developers to create smart contracts management system in different chains
EYWA is a cross-chain liquidity HUB and liquidity connector that gives developers access to all DeFi liquidity in one pool in any blockchain
EYWA is a cross-chain liquidity aggregator. One single interface aggregates all DEXes and provides users with instant access to all tokens in all chains: swap any assets between any chains.
What blockchains
do we support?
Ethereum, BNB Chain, Polygon, Fantom, Avalanche, Arbitrum, Cronos, Aurora (Hubchain).
What's
EYWA's fee?
A fee is charged by EYWA it in the following cases:
  1. When swapping tokens in both cross‑chain Stableswap and Uniswap pools
  2. When using a cross‑chain token bridge to transfer tokens;
  3. When using a Cross‑chain Data Protocol (CDP) to transfer data among blockchains.
What is EYWA's
roadmap?
You can find the EYWA's roadmap here. See a visual summary of completed steps and a vector of the EYWA project.
Will there be a token?
Yes. The EUSD token is a cross‑chain stablecoin, fully backed by native stablecoins from major blockchains
(Ethereum, BNB Chain, Polygon, Fantom, Avalanche, Arbitrum, Cronos, Aurora (Hubchain)).
What are the use
cases for EYWA?
  1. Transfer of stablecoins in different networks with the EYWA cross‑chain DEX;
  2. Cross‑chain farming EUSD;
  3. EYWA liquidity farming.

is backing EYWA?
